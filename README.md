## Actions

Usage:
```
make COMMAND [params ...]
 
commands:
    git-clone url=<GIT URL> branch=<GIT BRANCH>   
    git-checkout target=<BRANCH, TAG OR COMMIT HASH> is_hash=<0|1>   
    update-keys
    walter
    
default params values:
    is_hash 0
```
