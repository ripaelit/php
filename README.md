## Orchestration Actions

Usage:
```
make COMMAND [params ...]
 
commands:
    git-clone url [branch]
    git-checkout target [is_hash]   
    update-keys
    walter
    
default params values:
    is_hash 0
    branch "" Branch, tag or hash commit 
```
